# README — Sales Performance & Profitability Dashboard

## Project Overview

This project analyzes sales and profitability trends using SQL, Python, and Power BI. The analysis focuses on identifying business growth patterns, category-wise profitability, regional performance, and product-level sales-profit relationships through interactive visualizations and KPI-driven insights.

## Tools & Technologies Used

- SQL (SQLite3)
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI
- DAX

## Project Workflow

### 1) Data Cleaning & Preparation

- Removed unnecessary columns
- Handled missing values
- Converted date fields
- Prepared cleaned dataset for analysis

### 2) SQL Analysis

Performed SQL queries to analyze:

- Monthly sales and profit trends
- Region-wise performance
- Category and sub-category analysis
- Top-performing products
- Least profitable products

### 3) Python Exploratory Data Analysis (EDA)

Used Python libraries to perform:

- Sales distribution analysis
- Profit distribution analysis
- Outlier detection
- Sales vs Profit relationship analysis
- Trend analysis and visualizations

### 4) Power BI Dashboard

Created an interactive dashboard containing:

- Navigation page
- Sales Trend Analysis
- Category Analysis
- Regional Analysis
- Product Insights

## Dashboard Features

- Interactive slicers (Region, Month, Year)
- KPI cards for Sales, Profit, and Profit Margin
- Sales & Profit trend analysis
- Category-wise profitability analysis
- Regional contribution analysis
- Product-level sales vs profit analysis

## Key Business Insights

- Sales showed strong growth with recurring Q4 spikes.
- Technology category generated the highest profitability.
- Furniture category showed low profit margins despite high sales.
- West region contributed the highest overall profit.
- High sales did not always result in high profitability at product level.

## Conclusion
This project demonstrates an end-to-end business analytics workflow using SQL, Python, and Power BI to transform raw sales data into interactive dashboards and actionable business insights focused on growth, profitability, and operational efficiency.
